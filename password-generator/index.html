How to Use the Password Generator

  1. Step 1 — Drag the length slider to set your desired password length between 6 and 64 characters.
  2. Step 2 — Check or uncheck the character types you want: uppercase, lowercase, numbers, and/or symbols.
  3. Step 3 — Set the count, click "Generate Password", then copy individual passwords or use "Copy All" to copy all at once.

Features

  • Cryptographically secure randomness using the Web Crypto API
  • Customizable length from 6 to 64 characters
  • Toggle uppercase, lowercase, numbers, and symbols independently
  • Generate up to 20 passwords at once
  • Password strength indicator (Weak / Medium / Strong)
  • One-click copy for each password or all at once
  • 100% client-side — passwords never leave your browser

Frequently Asked Questions

How secure are the generated passwords? +
Passwords are generated using the browser's built-in Web Crypto API (crypto.getRandomValues), which produces cryptographically secure random numbers. This is the same standard used by security software and password managers.
Are my passwords stored or transmitted? +
No. All password generation happens locally in your browser. The passwords are never stored, logged, or transmitted to any server. Once you close or refresh the page, they are gone.
What password length should I use? +
For most online accounts, a minimum of 16 characters with a mix of all character types is recommended. For highly sensitive accounts like email or banking, use 20 or more characters.
What does the strength indicator measure? +
The strength meter evaluates length and character variety. A "Strong" password is at least 12 characters long and uses a mix of uppercase, lowercase, numbers, and symbols. "Weak" passwords are short or use only one character type.
Can I generate multiple passwords at once? +
Yes. Set the Count field to any number between 1 and 20, then click Generate. All passwords use the same settings. Use "Copy All" to copy all generated passwords to your clipboard at once.
