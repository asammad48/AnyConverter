Crop PDF Pages

Trim the margins or borders of every PDF page by entering crop values in PDF points (pt). Useful for removing unwanted whitespace or scanner borders.

Drop your PDF here or click to browse

Enter how many PDF points (pt) to crop from each edge. 1 pt ≈ 0.35 mm. A4 is 595×842 pt.

How to Crop a PDF

  1. Step 1 — Upload your PDF. The first page size is shown to help you calibrate.
  2. Step 2 — Enter the number of points to remove from each edge.
  3. Step 3 — Click "Crop & Download" to save the cropped PDF.

Features

  • Crops all pages uniformly by specified margins
  • Sets both CropBox and MediaBox for maximum compatibility
  • Works in PDF points (72 pt = 1 inch)
  • First page dimensions shown after upload to guide values
  • 100% browser-based processing

Frequently Asked Questions

What unit are the crop values in? +
PDF points (pt). There are 72 points per inch, so 1 cm ≈ 28 pt. An A4 page is 595×842 pt.
Will cropping reduce the file size? +
CropBox/MediaBox changes only affect what viewers display, not the underlying content. For a true file-size reduction, use the Optimize PDF tool.
