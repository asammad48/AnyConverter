Frequently Asked Questions

What is a cron expression? +
A cron expression is a 5-field string (minute hour day-of-month month day-of-week) used to schedule recurring tasks in Unix/Linux systems. Each field accepts numbers, wildcards (*), ranges (1-5), steps (*/5), or lists (1,3,5).
What does */5 mean? +
The /5 step notation means "every 5 units". So */5 in the minute field runs every 5 minutes (0, 5, 10, 15…). In the hour field, */2 would run every 2 hours.
What does 0 0 * * 1-5 mean? +
This runs at midnight (00:00) on every day of the week from Monday (1) to Friday (5), i.e., every weekday at midnight.
Can cron run multiple times per hour? +
Yes. Use */N in the minute field to run every N minutes. For example, */15 runs at 0, 15, 30, and 45 minutes past each hour.
Are seconds supported? +
Standard 5-field cron does not include seconds. Some systems use 6-field cron with a leading seconds field, but this parser follows the POSIX 5-field standard.
