How to Convert HTML to PDF

  1. Step 1 — Paste your HTML code into the editor (a sample is pre-loaded).
  2. Step 2 — Preview updates live as you type.
  3. Step 3 — Choose portrait or landscape orientation.
  4. Step 4 — Click "Convert to PDF & Download".

Features

  • Live HTML preview in an iframe
  • Multi-page A4 output — long content is split across pages
  • Portrait and landscape orientation
  • Supports inline CSS, tables, images (base64) and web fonts
  • Powered by jsPDF + html2canvas — fully browser-based

Frequently Asked Questions

Can I use external CSS or Google Fonts? +
External stylesheets and web fonts from external domains are blocked in the iframe sandbox. Use inline <style> tags for CSS and base64-encoded fonts for custom typography.
Can I convert a web page URL to PDF? +
Not directly. You can use your browser's "View Page Source" to copy the HTML, paste it here, and convert. However external resources like CSS and images won't load unless inlined.
