Frequently Asked Questions

Is my IP address stored when I use this tool? +
No. This tool makes a direct API call from your browser to ipapi.co. AnyConverter does not log or store any IP addresses you look up.
What does the ASN number mean? +
ASN stands for Autonomous System Number. It identifies the network (typically an ISP or large organization) that owns a block of IP addresses. For example, AS15169 is Google's network.
Can I look up IPv6 addresses? +
This tool supports IPv4 addresses and the API also handles most IPv6 addresses. Results may be less detailed for IPv6 depending on the available geolocation data.
Why is the location sometimes inaccurate? +
IP geolocation is based on registration data and network routing, not GPS. It's accurate to city/region level in most cases, but actual user may differ due to VPNs, proxies, or ISP routing.
