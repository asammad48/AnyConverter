    How to Use the Stopwatch

    1. Step 1 — Click Start to begin timing.
    2. Step 2 — Click Lap to record a split time without stopping.
    3. Step 3 — Click Start again (now says Pause) to pause the timer.
    4. Step 4 — Click Reset to clear all times and laps.

    Features

    • Millisecond precision using requestAnimationFrame and Date.now()
    • Unlimited lap recording — shows lap time and cumulative total
    • Start, pause, resume and reset controls
    • 100% browser-based — no data is sent anywhere

    Frequently Asked Questions

    Does the stopwatch keep running if I switch tabs? +
    Yes. The timer is based on Date.now() timestamps, so it remains accurate even when the tab is in the background. The display will update when you return to the tab.
    How many laps can I record? +
    There is no hard limit. You can record as many laps as you like — the list scrolls if it overflows the container.
