Frequently Asked Questions

What is a dead pixel vs a stuck pixel? +
A dead pixel is permanently off (appears black on any background). A stuck pixel is permanently on in one color (e.g., always red). Dead pixels typically cannot be fixed; stuck pixels sometimes recover with pixel-refreshing tools or pressure techniques.
How many dead pixels is acceptable? +
Monitor manufacturers use ISO 13406-2 standard which allows Class II monitors to have up to 2 completely dead pixels. However, many retailers accept returns for any dead pixels, especially in the center of the screen.
