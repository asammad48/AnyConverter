Frequently Asked Questions

Why use REM instead of pixels? +
REM (root em) units scale relative to the root element's font size. This makes layouts more accessible — users who increase their browser's font size will see your content scale proportionally, unlike fixed pixel values.
What base font size should I use? +
The browser default is 16px. Most projects keep this default, meaning 1rem = 16px. Some projects set the root to 62.5% (10px) to make the math easier (1rem = 10px, so 1.6rem = 16px). Match the base to your CSS :root declaration.
Is REM the same as EM? +
No. EM is relative to the current element's font size, which can compound in nested elements. REM always refers to the root element's font size, making it more predictable and consistent.
