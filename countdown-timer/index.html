How to Use the Countdown Timer

  1. Step 1 — Click a quick preset (5 min, 10 min, 25 min, 1 hour) or enter a custom time.
  2. Step 2 — Click Start to begin the countdown.
  3. Step 3 — Click Start again (now says Pause) to pause, then Resume to continue.
  4. Step 4 — An audio beep plays when the timer reaches zero.

Features

  • Quick presets: 5 min, 10 min, 25 min (Pomodoro), 1 hour
  • Custom time input up to 99 hours
  • Visual progress bar that shrinks as time passes
  • Audio alert using Web Audio API — no files needed
  • Pause and resume at any time

Frequently Asked Questions

Will the alarm sound if the tab is in the background? +
The timer will fire on time, but browsers may delay or silence audio on inactive tabs. Keep the tab visible for reliable audio. The countdown itself remains accurate regardless.
What is a Pomodoro timer? +
The Pomodoro Technique uses 25-minute focused work sessions followed by 5-minute breaks. Use the "25 min" preset for a classic Pomodoro session.
