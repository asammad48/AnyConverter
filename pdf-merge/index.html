    Frequently Asked Questions

    Is there a file size limit? +
    There is no hard file size limit, but very large PDFs may be slow to process depending on your device's memory. For best performance, keep total file size under 100MB.
    Are my PDF files secure? +
    Yes. Your PDF files never leave your device. All merging happens locally in your browser using the pdf-lib JavaScript library. Nothing is uploaded to any server.
    Can I merge password-protected PDFs? +
    Password-protected PDFs cannot be merged without the password. You would need to remove the password protection first using a PDF editor.
    How many PDF files can I merge at once? +
    You can merge as many PDFs as your browser's memory allows. Typically 10–20 average-sized documents work smoothly. Larger numbers may take more time.
    Will the quality of my PDFs be affected? +
    No. pdf-lib copies pages from source PDFs without re-compressing or altering them, so the quality of text, images, and vector graphics is preserved exactly.
