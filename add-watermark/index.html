How to Add a Watermark to a PDF

  1. Step 1 — Upload your PDF file.
  2. Step 2 — Type your watermark text (e.g. "CONFIDENTIAL").
  3. Step 3 — Choose position, size, opacity and colour.
  4. Step 4 — Click "Add Watermark & Download".

Features

  • 4 position options: diagonal, centre, top, bottom
  • Adjustable opacity (5%–100%)
  • Custom colour picker
  • Font size from 10pt to 120pt
  • 100% browser-based — no server upload

Frequently Asked Questions

Can I add an image watermark? +
Currently this tool supports text watermarks only. For image watermarks you can use the Edit PDF tool to overlay an image manually.
Will the watermark be permanent? +
Yes. The watermark is drawn directly onto the page content and cannot be removed without specialised PDF editing software.
