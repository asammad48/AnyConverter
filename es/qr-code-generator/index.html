Generador de Códigos QR

Crea códigos QR al instante para cualquier texto, URL, correo electrónico o número de teléfono. Elige el tamaño y el nivel de corrección de errores, luego descarga o copia el PNG. Todo se ejecuta en tu navegador, nada se sube a ningún servidor.

Cómo Usar el Generador de Códigos QR

  1. Paso 1 — Introduce cualquier texto, URL, dirección de correo electrónico o número de teléfono en el campo de entrada.
  2. Paso 2 — Selecciona el tamaño preferido (128, 256 o 512 píxeles) y el nivel de corrección de errores.
  3. Paso 3 — Haz clic en "Generar Código QR", luego descarga el PNG o cópialo al portapapeles.

Características

  • Genera códigos QR para texto, URLs, correos, teléfonos y más
  • Tres opciones de tamaño: 128×128, 256×256 y 512×512 píxeles
  • Tres niveles de corrección de errores: Baja, Media y Alta
  • Descarga el código QR como imagen PNG con un clic
  • Copia la imagen del código QR directamente al portapapeles
  • 100% en el navegador — ningún dato se sube a ningún servidor
  • Pulsa Enter para generar al instante desde el teclado

Preguntas Frecuentes

¿Qué puedo codificar en un código QR? +
Puedes codificar cualquier cadena de texto, incluyendo URLs de sitios web, direcciones de correo electrónico, números de teléfono, mensajes de texto sin formato, credenciales de Wi-Fi y mucho más.
¿Qué significa el nivel de corrección de errores? +
La corrección de errores permite leer un código QR incluso cuando está parcialmente dañado. Baja (L) corrige hasta el 7% de daño, Media (M) hasta el 15%, y Alta (H) hasta el 30%.
¿Se envían mis datos a un servidor? +
No. Toda la generación de códigos QR ocurre en tu navegador usando JavaScript. Tu texto o URL nunca se transmite a ningún servidor externo.
¿Qué tamaño de código QR debo usar? +
Para uso web, 256×256 es una buena opción por defecto. Para imprimir en folletos o tarjetas de visita, usa 512×512 para asegurarte de que el código se pueda escanear correctamente.
¿Puedo usar el código QR generado comercialmente? +
Sí. Los códigos QR generados con esta herramienta son gratuitos para uso personal y comercial. No hay marcas de agua ni restricciones en cómo usas las imágenes PNG descargadas.
