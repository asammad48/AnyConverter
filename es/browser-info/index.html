Ve todo sobre tu navegador, sistema operativo, pantalla y hardware en un solo lugar — nombre del navegador, versión, idioma, resolución de pantalla, núcleos de CPU, zona horaria y más.

Cómo Usar el Visor de Info del Navegador

  1. La tabla se carga automáticamente al abrir la página — no se necesita ninguna entrada.
  2. Desplázate por más de 20 filas para ver cada dato.
  3. Haz clic en Copiar Todo para copiar toda la información como texto plano.

Características

  • Más de 20 datos: navegador, SO, idioma, pantalla, ventana, DPR, CPU, memoria
  • Detección de soporte de WebGL, localStorage y Service Worker
  • Tipo de conexión mediante la API de Información de Red
  • Detección de zona horaria
  • Copia toda la información con un clic

Preguntas Frecuentes

¿Es privada esta información? +
Toda la información se lee directamente de las APIs JavaScript de tu navegador y se muestra localmente. Nada se envía a ningún servidor. Sin embargo, estos son los mismos datos a los que puede acceder cualquier sitio web que visites.
¿Por qué muestra el navegador incorrecto? +
Algunos navegadores modifican su cadena User Agent para coincidir con otro navegador por compatibilidad. Los navegadores basados en Chromium como Brave pueden aparecer como Chrome. La detección se basa en el análisis de User Agent, que puede ser impreciso.
