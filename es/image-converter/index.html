Preguntas Frecuentes

¿Se suben mis imágenes a un servidor? +
No. La conversión se realiza completamente en tu navegador usando Canvas API. Tus imágenes nunca abandonan tu dispositivo.
¿Qué calidad debo elegir para JPG? +
Para la mayoría de usos, 80-85% ofrece un buen equilibrio entre calidad y tamaño de archivo. Para imágenes que se usarán en imprenta, usa 90-95%.
¿Por qué usar WebP en lugar de JPG? +
WebP suele producir archivos entre un 25-35% más pequeños que JPG con calidad visual equivalente, lo que lo hace ideal para imágenes web.
