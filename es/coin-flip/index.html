Cómo lanzar la moneda

  1. Paso 1 — Haz clic en Lanzar Moneda para tirar.
  2. Paso 2 — Observa la animación y ve el resultado (Cara o Cruz).
  3. Paso 3 — Los contadores y el historial se actualizan automáticamente.
  4. Paso 4 — Haz clic en Reiniciar para borrar el recuento y el historial.

Características

  • Animación de moneda 3D en CSS
  • Resultado aleatorio 50/50 usando Math.random()
  • Contadores acumulados de cara y cruz
  • Últimos 20 lanzamientos mostrados como fichas de historial de colores
  • Botón de reinicio que borra todas las estadísticas

Preguntas Frecuentes

¿Es el lanzamiento realmente 50/50? +
Sí. El resultado se determina mediante Math.random() < 0.5. El generador de números pseudoaleatorios de JavaScript produce una distribución esencialmente uniforme, dando a cada resultado un 50% de probabilidad.
¿Puedo lanzar varias monedas a la vez? +
No en esta página — está diseñada para lanzamientos individuales. Para múltiples lanzamientos simultáneos, prueba el Lanzador de Dados con un d2.
