Convertidor JSON ↔ YAML

Convierte entre JSON y YAML al instante. Pega tu JSON para obtener una salida YAML limpia, o convierte YAML de vuelta a JSON formateado. Valida la sintaxis mientras escribes.

Cómo Usar el Convertidor JSON a YAML

  1. Elige la dirección de conversión: JSON→YAML o YAML→JSON usando los botones de modo.
  2. Pega tu JSON o YAML en el área de entrada (o haz clic en "Cargar Ejemplo" para ver una muestra).
  3. La salida convertida aparece al instante. Haz clic en "Copiar Salida" para copiar el resultado.

Características

  • Conversión bidireccional — JSON a YAML y YAML a JSON
  • Conversión en tiempo real mientras escribes
  • Validación de sintaxis con mensajes de error claros
  • Sangría de 2 espacios en la salida YAML
  • Carga datos de ejemplo con un clic
  • 100% del lado del cliente — tus datos nunca salen del navegador

Preguntas Frecuentes

¿Qué es YAML?

YAML (YAML Ain't Markup Language) es un formato de serialización de datos amigable para humanos. Usa sangría y puntuación mínima, lo que lo hace más legible que JSON. Es ampliamente usado para archivos de configuración (Docker, Kubernetes, GitHub Actions).

¿Todo JSON es YAML válido?

Sí — JSON es técnicamente un subconjunto de YAML 1.2. Cualquier documento JSON válido también es YAML válido. Sin embargo, YAML admite funciones adicionales como comentarios y cadenas multilínea que JSON no tiene.

¿YAML admite comentarios?

Sí. YAML admite comentarios usando el carácter #. JSON no admite comentarios. Al convertir YAML a JSON, los comentarios se descartan.

¿Qué sangría usa el convertidor?

El convertidor genera YAML con sangría de 2 espacios, que es la convención más común para archivos YAML en las herramientas modernas.

¿Mis datos se envían a un servidor?

No. Esta herramienta funciona completamente en tu navegador usando la biblioteca js-yaml. Tus datos nunca se suben a ningún servidor, lo que la hace segura para datos de configuración sensibles.
