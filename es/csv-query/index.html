Cómo consultar un archivo CSV con SQL

  1. Sube tu CSV — Arrastra o selecciona un archivo CSV con fila de encabezado. Se carga en una base de datos SQLite del navegador como tabla llamada data.
  2. Escribe SQL — Consulta con SQL completo: SELECT, WHERE, GROUP BY, ORDER BY, LIMIT, JOINs, agregaciones. Usa Ctrl+Enter para ejecutar rápido.
  3. Explora y exporta — Ordena los resultados haciendo clic en los encabezados de columna. Navega por las páginas. Exporta el conjunto completo de resultados como CSV o JSON.

Características

  • SQL completo de SQLite — SELECT, WHERE, GROUP BY, ORDER BY, agregaciones
  • Resultados paginados — 50 filas por página
  • Columnas ordenables — haz clic en los encabezados para ordenar asc/desc
  • Exporta resultados como CSV o JSON
  • Historial de consultas — últimas 5 consultas, haz clic para restaurar
  • 100% basado en el navegador — con SQL.js (SQLite WASM)

Preguntas frecuentes

¿Qué sintaxis SQL es compatible? +
La herramienta usa SQL.js, una versión de SQLite compilada a WebAssembly. Es compatible con SELECT, WHERE, GROUP BY, ORDER BY, LIMIT, OFFSET, JOINs, subconsultas y agregaciones como COUNT, SUM, AVG, MIN y MAX. La mayoría de las funciones de SQLite también funcionan.
¿Cuál es el nombre de la tabla para mi CSV? +
Tu CSV se carga en una tabla llamada data. Los nombres de columna provienen de la primera fila (fila de encabezado) del CSV. Si un nombre de columna tiene espacios, ponlo entre comillas dobles: SELECT "nombre apellido" FROM data.
¿Qué tan grande puede ser el archivo que consulto? +
No hay un límite estricto — la memoria disponible del navegador es la restricción. Los archivos de hasta 50 MB suelen funcionar bien. Los archivos muy grandes pueden tardar un momento en cargarse en la base de datos SQLite en memoria.
¿Se envían mis datos a un servidor? +
No. Todo el procesamiento se ejecuta localmente en tu navegador. SQL.js es una versión WASM de SQLite que funciona completamente en el lado del cliente. Tus datos CSV y los resultados de las consultas nunca abandonan tu dispositivo.
¿Puedo ejecutar múltiples consultas sobre el mismo archivo? +
Sí. Una vez cargado un CSV, puedes ejecutar tantas consultas como quieras. El historial guarda tus últimas 5 consultas para reutilizarlas rápidamente. Sube un nuevo archivo para reemplazar el conjunto de datos actual.
