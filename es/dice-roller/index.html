Lanzador de Dados Online

Lanza cualquier combinación de dados poliédricos — d4, d6, d8, d10, d12 o d20. Lanza hasta 20 dados a la vez y ve el total. Perfecto para juegos de rol de mesa y juegos de mesa.

Cómo lanzar dados

  1. Paso 1 — Haz clic en un botón de tirada rápida (d4 a d20) para una tirada simple instantánea.
  2. Paso 2 — O configura el número de dados y caras en los campos personalizados y haz clic en ¡Lanzar!
  3. Paso 3 — Cada resultado del dado se muestra como una ficha. El total aparece debajo para múltiples dados.
  4. Paso 4 — Las últimas 10 tiradas se muestran en la sección de historial.

Características

  • Botones de tirada rápida para todos los dados poliédricos estándar (d4–d20)
  • Dados personalizados: cualquier número de caras del 2 al 100
  • Lanza hasta 20 dados a la vez con total sumado
  • Resultados del d6 mostrados como caras con puntos ⚀–⚅
  • Últimas 10 tiradas registradas en el historial

Preguntas frecuentes

¿Puedo lanzar dados no estándar como d3 o d100? +
Sí. Usa los campos de tirada personalizada e ingresa cualquier número de caras entre 2 y 100. Un d100 (dado percentil) se introduce como 100 caras.
¿Es bueno para D&D u otros juegos de rol? +
Absolutamente. Todos los dados estándar de juegos de rol están disponibles como botones de tirada rápida. También puedes lanzar varios dados a la vez para simular tiradas de daño como 2d6+3 (lanza 2 dados, suma 3 manualmente).
