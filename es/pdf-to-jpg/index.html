Convertidor PDF a JPG

Renderiza cada página de tu PDF como una imagen de alta calidad. Elige el formato de salida (JPEG o PNG), la escala de resolución y la calidad, previsualiza las miniaturas y descarga todas las imágenes.

Cómo Convertir PDF a JPG

  1. Paso 1 — Sube tu PDF. Las páginas se renderizan automáticamente.
  2. Paso 2 — Ajusta el formato, la escala y la calidad si es necesario, luego haz clic en "Volver a renderizar".
  3. Paso 3 — Previsualiza las miniaturas y haz clic en "Descargar Todas las Imágenes".

Características

  • Renderiza cada página en un archivo de imagen separado
  • Formatos de salida JPEG y PNG
  • 3 escalas de resolución: 1×, 2×, 3× (72–216 DPI)
  • Calidad JPEG ajustable
  • Vista previa en miniatura antes de descargar
  • 100% en el navegador mediante PDF.js

Preguntas Frecuentes

¿Qué escala debo usar para imágenes de calidad de impresión? +
Usa 3× para obtener calidad de impresión (~216 DPI). Para uso web, 2× es suficiente. 1× produce imágenes de resolución de pantalla útiles para vistas previas.
¿Puedo convertir solo una página? +
La herramienta convierte todas las páginas, pero puedes elegir guardar solo las imágenes de páginas específicas. Como alternativa, usa primero la herramienta de extracción de páginas para crear un PDF de una sola página y conviértelo aquí.
