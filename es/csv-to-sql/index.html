Cómo convertir CSV a SQL

  1. Introduce el CSV — Pega los datos CSV o sube un archivo .csv. Elige el delimitador si es necesario.
  2. Configura los tipos de columna — Revisa los tipos de columna detectados automáticamente (INT, VARCHAR, DATE, etc.) y ajústalos si es necesario.
  3. Genera el SQL — Define el nombre de tabla, elige el dialecto y haz clic en "Generar SQL". Copia o descarga el resultado.

Características

Preguntas Frecuentes

¿Qué dialectos SQL están disponibles? +
MySQL (comillas invertidas y AUTO_INCREMENT), PostgreSQL (identificadores entre comillas dobles, SERIAL) y SQLite (INTEGER PRIMARY KEY AUTOINCREMENT).
¿Cómo se gestionan las celdas vacías? +
Las celdas vacías en el CSV se convierten en NULL en las sentencias INSERT SQL, que es la representación estándar de un valor ausente en SQL.
¿Se escapan las comillas simples en los datos? +
Sí. Las comillas simples en los valores de texto se escapan duplicándolas (por ejemplo, O'Brien se convierte en O''Brien), que es el escapado estándar de cadenas en SQL.
