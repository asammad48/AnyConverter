Preguntas Frecuentes

      

        
¿Qué es Markdown? +
Markdown es un lenguaje de marcado ligero que usa sintaxis de formato de texto plano. Fue diseñado para ser fácil de escribir y leer, y se convierte limpiamente a HTML.

        
¿Qué es GitHub Flavored Markdown? +
GitHub Flavored Markdown (GFM) es un superconjunto del Markdown estándar que añade funciones como tablas, listas de tareas, texto tachado y bloques de código delimitados con resaltado de sintaxis.

        
¿Cómo añado un enlace en Markdown? +
Usa la sintaxis [texto del enlace](URL). Por ejemplo, [AnyConverter](https://anyconverter.io) crea un enlace clicable.

        
¿Cómo añado una imagen en Markdown? +
Usa ![texto alternativo](url-imagen) — similar a un enlace pero con un signo de exclamación al principio. El texto alternativo se muestra si la imagen no carga.

        
¿Puedo usar la salida HTML en mi web? +
Sí. El HTML generado es limpio y está listo para pegar en cualquier página web. Puede que necesites añadir tus propios estilos CSS, ya que esta herramienta no incluye una hoja de estilos en el archivo descargado.
