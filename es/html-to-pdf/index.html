Conversor de HTML a PDF

Pega código HTML en el editor, previsualízalo en vivo en el navegador, y luego conviértelo en un PDF A4 de varias páginas. Soporta CSS en línea, tablas, encabezados e imágenes en base64.

Publicidad

Cómo Convertir HTML a PDF

  1. Paso 1 — Pega tu código HTML en el editor (hay un ejemplo precargado).
  2. Paso 2 — La vista previa se actualiza en vivo mientras escribes.
  3. Paso 3 — Elige orientación vertical u horizontal.
  4. Paso 4 — Haz clic en "Convertir a PDF y Descargar".

Características

  • Vista previa HTML en vivo en un iframe
  • Salida A4 de varias páginas — el contenido largo se divide entre páginas
  • Orientación vertical y horizontal
  • Soporta CSS en línea, tablas, imágenes (base64) y fuentes web
  • Desarrollado con jsPDF + html2canvas — completamente en el navegador

Preguntas Frecuentes

¿Puedo usar CSS externo o Google Fonts? +
Las hojas de estilo externas y fuentes web de dominios externos están bloqueadas en el sandbox del iframe. Usa etiquetas <style> en línea para CSS y fuentes codificadas en base64 para tipografía personalizada.
¿Puedo convertir una URL de página web a PDF? +
No directamente. Puedes usar "Ver código fuente" del navegador para copiar el HTML, pegarlo aquí y convertirlo. Sin embargo, los recursos externos como CSS e imágenes no se cargarán a menos que estén en línea.
