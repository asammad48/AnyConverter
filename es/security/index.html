Seguridad en AnyConverter

Última actualización: 26 de mayo de 2026

La seguridad y la privacidad son fundamentales en la forma en que AnyConverter está construido — no son añadidos de último momento. Así es como protegemos tus datos y nuestra plataforma.

🔒

Sin Subida de Archivos

Todo el procesamiento de herramientas ocurre en tu navegador. Los archivos nunca llegan a nuestros servidores.

🔐

HTTPS en Todo Momento

Cada página se sirve mediante TLS. Todas las conexiones están cifradas en tránsito.

🚫

Sin Cuentas

Sin nombres de usuario, contraseñas ni tokens de sesión. Nada que robar.

🛡️

Protección CDN

Servido a través de Cloudflare con protección DDoS y caché de borde.

Arquitectura del Lado del Cliente

La propiedad de seguridad más importante de AnyConverter es que todo el procesamiento de herramientas ocurre dentro de tu navegador. Cuando usas nuestro combinador de PDF, formateador JSON, conversor de imágenes o cualquier otra herramienta, tus datos son procesados por JavaScript que se ejecuta en tu propio dispositivo.

Esto significa:

  • No podemos ver tus archivos, aunque quisiéramos
  • No existe una base de datos en el servidor que almacene tu contenido
  • Cerrar la pestaña del navegador elimina inmediatamente todos los rastros de tus datos
  • La mayoría de las herramientas siguen funcionando sin conexión después de que se carga la página

HTTPS y Seguridad en el Transporte

Todo el tráfico hacia y desde AnyConverter está cifrado mediante TLS 1.2 o superior. Usamos HTTP Strict Transport Security (HSTS) para prevenir ataques de degradación. Nuestros certificados SSL se gestionan automáticamente a través de Cloudflare.

Política de Seguridad de Contenido

AnyConverter implementa una estricta Política de Seguridad de Contenido (CSP) para prevenir ataques de scripting entre sitios (XSS). Solo cargamos scripts y recursos de fuentes de confianza (nuestro propio dominio, Google Fonts y bibliotecas CDN explícitamente permitidas).

Bibliotecas de Terceros

Cuando usamos bibliotecas JavaScript de terceros (pdf-lib, PapaParse, highlight.js), las cargamos desde CDNs de confianza con hashes de Integridad de Subrecursos (SRI) donde es posible, asegurando que el código no haya sido manipulado.

Sin Datos de Cuenta que Vulnerar

Dado que AnyConverter no tiene cuentas de usuario, no existe ninguna base de datos de nombres de usuario, correos electrónicos o contraseñas que pueda ser vulnerada. No podemos perder datos que nunca recopilamos.

Infraestructura

AnyConverter es un sitio web estático desplegado en Cloudflare Pages, una de las plataformas de alojamiento estático más seguras y eficientes disponibles. Cloudflare proporciona mitigación automática de DDoS, reglas WAF y caché de borde en más de 200 ubicaciones globales.

🔍 Divulgación Responsable

Si descubres una vulnerabilidad de seguridad en AnyConverter, te animamos a reportarla de forma responsable. Tomamos muy en serio todos los informes de seguridad y responderemos con prontitud.

Por favor, reporta los problemas de seguridad a: security@anyconverter.io

Incluye una descripción clara de la vulnerabilidad, los pasos para reproducirla y el impacto potencial. Te pedimos amablemente que no divulgues públicamente el problema hasta que hayamos tenido un tiempo razonable para investigar y responder (generalmente en un plazo de 72 horas).

Actualmente no ofrecemos un programa de recompensas por errores, pero reconoceremos públicamente tu contribución si así lo deseas.
