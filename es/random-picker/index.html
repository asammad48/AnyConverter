Selector Aleatorio Online

Pega una lista de elementos y selecciona aleatoriamente el número que quieras. Los elementos pueden estar separados por líneas o comas. Los elementos seleccionados se resaltan en el grupo.

Publicidad

Cómo Elegir Elementos Aleatorios

  1. Paso 1 — Pega tu lista de elementos en el área de texto (uno por línea o separados por comas).
  2. Paso 2 — Selecciona el separador que coincida con el formato de tu lista.
  3. Paso 3 — Establece cuántos elementos elegir.
  4. Paso 4 — Haz clic en ¡Elegir! para ver los elementos seleccionados resaltados en el grupo.

Características

  • Separador por nueva línea o coma
  • Elige cualquier número de elementos (hasta el tamaño de la lista)
  • Elementos seleccionados mostrados como etiquetas
  • Grupo completo visible con los elementos seleccionados resaltados
  • Usa el algoritmo Fisher-Yates para aleatoriedad uniforme

Preguntas Frecuentes

¿Son posibles los duplicados? +
No. El selector usa un algoritmo de barajado que garantiza que cada elemento solo pueda ser seleccionado una vez por elección. Si tu lista contiene entradas duplicadas, cada una puede ser seleccionada de forma independiente.
¿Qué tan grande puede ser mi lista? +
No hay límite fijo. Las listas con cientos de elementos funcionan bien ya que todo el procesamiento se realiza en el navegador con JavaScript.
