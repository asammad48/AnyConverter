Preguntas Frecuentes

¿Para qué sirve un hash? +
Los hashes se usan para verificar la integridad de archivos, almacenar contraseñas de forma segura, generar identificadores únicos y detectar duplicados o cambios en datos.
¿Es MD5 seguro para contraseñas? +
No. MD5 está obsoleto para uso criptográfico. Para contraseñas, usa bcrypt, scrypt o Argon2. MD5 sigue siendo útil para verificar integridad de archivos no críticos.
¿Puedo hashear archivos grandes? +
Sí. La herramienta lee el archivo en fragmentos para que no se sature la memoria. Archivos de varios GB pueden procesarse aunque puede tardar unos segundos.
