Convertidor de Código Morse

Convierte texto a código Morse y viceversa al instante. Soporta las 26 letras, dígitos 0-9 y puntuación común. Usa barra diagonal (/) para separar palabras al decodificar. Gratis, sin registro.

Cómo Usar el Convertidor de Código Morse

  1. Paso 1 — Elige "Texto → Morse" para codificar o "Morse → Texto" para decodificar.
  2. Paso 2 — Escribe o pega tu entrada en el panel izquierdo. La conversión ocurre mientras escribes.
  3. Paso 3 — Haz clic en "Copiar salida" para copiar el código Morse o el texto decodificado.

Características

  • Conversión bidireccional — texto a Morse y Morse a texto
  • Soporta A-Z, 0-9 y puntuación común (., , ? ! / : ; = + @ $)
  • Separador de palabras: barra diagonal (/) entre palabras Morse
  • Conversión en tiempo real
  • Copia con un clic
  • Funciona completamente en el navegador

Preguntas Frecuentes

¿Cómo decodifico el código Morse? +
Haz clic en "Morse → Texto" e introduce puntos (.) y rayas (-) separados por espacios entre las letras. Usa una barra diagonal ( / ) para separar palabras. Por ejemplo: ".... . .-.. .-.. --- / .-- --- .-. .-.. -.." se decodifica como "HELLO WORLD".
¿Qué caracteres son compatibles? +
Las 26 letras del abecedario (A-Z), dígitos 0-9 y puntuación común: punto, coma, signo de interrogación, exclamación, barra, paréntesis, ampersand, dos puntos, punto y coma, igual, más, guion bajo, comillas, dólar y arroba.
¿Por qué la salida está en mayúsculas? +
El código Morse no distingue mayúsculas de minúsculas — no existen símbolos Morse separados para letras mayúsculas y minúsculas. La convención estándar es decodificar Morse en mayúsculas, aunque puedes cambiar el formato después si es necesario.
¿Qué significa "/" en Morse? +
En esta herramienta, una barra diagonal rodeada de espacios ( / ) representa el separador de palabras. Al codificar, los espacios entre palabras se convierten en " / ". Al decodificar, usa " / " para separar palabras en tu entrada Morse.
