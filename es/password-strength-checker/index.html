Cómo Usar el Comprobador de Fortaleza de Contraseña

  1. Paso 1 — Escribe tu contraseña en el campo de entrada.
  2. Paso 2 — La barra de fortaleza y la etiqueta se actualizan instantáneamente mientras escribes, mostrando Débil/Regular/Buena/Fuerte.
  3. Paso 3 — Revisa la lista de verificación para ver qué criterios cumple tu contraseña y mejórala según corresponda.

Características

  • Calificación de fortaleza en tiempo real: Débil, Regular, Buena, Fuerte
  • Lista de verificación de 7 puntos (longitud, mayúsculas, minúsculas, dígitos, caracteres especiales, no común)
  • Barra de fortaleza con código de colores (rojo/ámbar/azul/verde)
  • Botón Mostrar/Ocultar para el campo de contraseña
  • Comprueba contra una lista de contraseñas más comunes
  • 100% en el navegador — la contraseña nunca se envía a ningún servidor

Preguntas Frecuentes

¿Qué hace que una contraseña sea fuerte? +
Una contraseña fuerte tiene al menos 12 caracteres, incluye letras mayúsculas y minúsculas, números y caracteres especiales, y no es una palabra o frase de uso común. Esta herramienta comprueba todos estos criterios simultáneamente.
¿Se almacena mi contraseña en algún lugar? +
No. Esta herramienta se ejecuta completamente en tu navegador usando JavaScript. Tu contraseña nunca se envía a ningún servidor y nunca se almacena en ningún lugar. Incluso puedes desconectarte de internet antes de escribir tu contraseña.
¿Qué cuenta como carácter especial? +
Cualquier carácter que no sea una letra (a-z, A-Z) o dígito (0-9) se considera especial. Ejemplos: ! @ # $ % ^ & * ( ) _ - + = [ ] { } | \ : ; " ' < > , . ? /
¿Qué significa "contraseña común"? +
La herramienta comprueba tu contraseña contra una lista de las contraseñas más utilizadas (como "password", "123456", "qwerty"). Usar una contraseña común es extremadamente arriesgado porque los atacantes siempre las prueban primero.
