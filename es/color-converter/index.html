Preguntas Frecuentes

¿Cuál es la diferencia entre HEX y RGB? +
HEX y RGB representan los mismos colores pero en notaciones diferentes. HEX usa un código hexadecimal de 6 dígitos (p. ej. #FF5733), mientras que RGB usa valores decimales para los canales Rojo, Verde y Azul (p. ej. rgb(255, 87, 51)).
¿Qué es HSL? +
HSL son las siglas de Matiz (Hue), Saturación y Luminosidad (Lightness). Es un modelo más intuitivo para las personas — piensas en términos de ángulo de color (0–360°), viveza (saturación) y claridad u oscuridad (luminosidad).
¿Para qué se usa CMYK? +
CMYK (Cian, Magenta, Amarillo, Negro) se usa en diseño de impresión. Mientras que las pantallas usan RGB (color aditivo), las impresoras usan CMYK (mezcla de color sustractivo).
¿Qué es HSB (también llamado HSV)? +
HSB (Matiz, Saturación, Brillo) —también llamado HSV (Value/Valor)— es similar a HSL pero usa Brillo/Valor en lugar de Luminosidad. Se usa en muchas aplicaciones de diseño como Photoshop.
¿Cómo encuentro el código HEX de un color en pantalla? +
Usa el selector de color integrado de tu sistema operativo, las herramientas de desarrollo del navegador (inspeccionar elemento → muestra de color) o herramientas de captura de pantalla. Luego pega el valor HEX aquí para convertirlo a otros formatos.
