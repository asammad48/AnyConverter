Seguridad PDF — Establecer Permisos

Aplica restricciones de permisos detalladas a un PDF: controla la impresión, copia, edición, anotaciones y relleno de formularios. Requiere una contraseña de propietario para bloquear la configuración.

Cómo Establecer Permisos en un PDF

  1. Paso 1 — Sube tu PDF.
  2. Paso 2 — Establece una contraseña de propietario (obligatoria) y una contraseña de usuario opcional.
  3. Paso 3 — Marca o desmarca los permisos que quieras permitir.
  4. Paso 4 — Haz clic en "Aplicar Seguridad y Descargar".

Características

  • Contraseña de propietario requerida para cambiar permisos
  • Contraseña de usuario opcional para restringir la apertura
  • 5 indicadores de permisos: imprimir, copiar, modificar, anotar, rellenar formularios
  • Compatible con todos los visores de PDF estándar
  • 100% en el navegador: sin subida al servidor

Preguntas Frecuentes

¿Cuál es la diferencia entre contraseña de propietario y contraseña de usuario? +
La contraseña de usuario es necesaria para abrir el PDF. La contraseña de propietario es necesaria para cambiar permisos o eliminar restricciones. Ambas se establecen aquí; solo la contraseña de propietario es obligatoria.
¿Puedo eliminar estas restricciones más tarde? +
Sí. Usa la herramienta Desbloquear PDF e introduce la contraseña de propietario para eliminar todas las restricciones y obtener una copia sin restricciones.
