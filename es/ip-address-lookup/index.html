Búsqueda de Dirección IP

Busca la geolocalización, ISP, zona horaria y detalles de red de cualquier dirección IP al instante. Deja el campo en blanco para ver tus propios datos de IP.

Cómo Usar la Búsqueda de Dirección IP

  1. Paso 1 — Deja el campo en blanco y haz clic en "Mi IP" para buscar tu propia dirección IP, o escribe cualquier dirección IPv4.
  2. Paso 2 — Haz clic en "Buscar" para obtener la geolocalización y datos de red.
  3. Paso 3 — Ve ciudad, región, país, coordenadas, zona horaria, ISP y más en el panel de resultados.

Características

  • Busca cualquier dirección IPv4 o tu propia IP con un clic
  • Muestra ciudad, región, país, código postal, coordenadas, zona horaria y UTC
  • ISP / organización y ASN
  • Información de moneda e idioma
  • Sin datos almacenados

Preguntas Frecuentes

¿Se almacena mi dirección IP cuando uso esta herramienta? +
No. Esta herramienta realiza una llamada directa a la API de ipapi.co desde tu navegador. AnyConverter no registra ni almacena ninguna dirección IP que consultes.
¿Qué significa el número ASN? +
ASN significa Número de Sistema Autónomo. Identifica la red (normalmente un ISP o una gran organización) que posee un bloque de direcciones IP. Por ejemplo, AS15169 es la red de Google.
¿Puedo buscar direcciones IPv6? +
Esta herramienta soporta direcciones IPv4 y la API también gestiona la mayoría de las direcciones IPv6. Los resultados pueden ser menos detallados para IPv6 dependiendo de los datos de geolocalización disponibles.
¿Por qué la ubicación a veces es imprecisa? +
La geolocalización por IP se basa en datos de registro y enrutamiento de red, no en GPS. Es precisa a nivel de ciudad/región en la mayoría de los casos, pero el usuario real puede diferir por el uso de VPNs, proxies o el enrutamiento del ISP.
