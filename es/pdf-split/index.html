Cómo Dividir un PDF

  1. Sube tu PDF — Arrastra el archivo PDF o haz clic para seleccionarlo. La herramienta muestra el número total de páginas.
  2. Elige el modo de división — Extrae páginas específicas por rango, divide cada N páginas o extrae cada página por separado.
  3. Descarga — Haz clic en Dividir PDF y descarga cada archivo resultante.

Características

  • Extrae páginas específicas por rango
  • Divide cada N páginas
  • Extrae cada página por separado

  • 100% privado: sin subida al servidor
  • Basado en pdf-lib

Preguntas Frecuentes

¿Cómo especifico rangos de páginas? +
Usa comas para separar páginas individuales y guiones para rangos. Ejemplo: 1,3,5-8,12 extrae las páginas 1, 3, 5, 6, 7, 8 y 12.
¿Puedo extraer una sola página de un PDF? +
Sí. Usa el modo "Extraer páginas específicas" e introduce solo el número de página que deseas (ej. "5" para obtener únicamente la página 5).
¿Se suben mis PDFs a un servidor? +
No. Toda la división ocurre localmente en tu navegador usando la biblioteca JavaScript pdf-lib. Tus archivos nunca abandonan tu dispositivo.
¿Qué ocurre con la calidad de las páginas al dividir? +
pdf-lib copia las páginas sin reprocesarlas, por lo que la calidad se preserva perfectamente. El texto, las imágenes y los gráficos son idénticos al original.
¿Puedo dividir un PDF grande? +
Sí. No hay un límite de tamaño estricto. Los PDFs muy grandes pueden tardar un momento en procesarse en el navegador, pero toda la división es del lado del cliente sin ninguna subida.
