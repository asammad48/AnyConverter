Lista de Tareas Online

Una lista de tareas sencilla y persistente que se guarda automáticamente en tu navegador. Sin cuenta, sin sincronización, sin complicaciones — solo tareas.

    Publicidad

    Cómo Usar la Lista de Tareas

    1. Paso 1 — Escribe una tarea en el campo de entrada y haz clic en Añadir o pulsa Intro.
    2. Paso 2 — Marca la casilla junto a una tarea para marcarla como completada.
    3. Paso 3 — Usa los botones de filtro para ver Todas, Activas o Completadas.
    4. Paso 4 — Haz clic en Eliminar completadas para borrar todas las tareas hechas de una vez.

    Características

    • Se guarda en localStorage — las tareas persisten entre sesiones
    • Filtra por Todas, Activas o Completadas
    • Elimina las tareas completadas con un clic
    • El contador de elementos muestra las tareas activas restantes
    • Funciona sin conexión — no se necesita internet después de cargar la página

    Preguntas Frecuentes

    ¿Se guardarán mis tareas si cierro el navegador? +
    Sí. Las tareas se almacenan en el localStorage de tu navegador y persisten entre sesiones en el mismo dispositivo y navegador. Borrar los datos del navegador o usar el modo privado/incógnito las eliminará.
    ¿Puedo sincronizar las tareas entre dispositivos? +
    Por ahora no — el localStorage es específico del dispositivo y el navegador. Para sincronización entre dispositivos necesitarías un servicio basado en cuenta.
