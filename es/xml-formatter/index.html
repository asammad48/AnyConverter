Cómo formatear XML

  1. Pega tu XML — Pega el contenido XML en el área de texto o sube un archivo .xml.
  2. Formatea o valida — Haz clic en "Formatear" para añadir sangría o en "Validar" para comprobar errores de sintaxis.
  3. Copia o descarga — Copia el resultado o descárgalo como archivo .xml.

Características

  • Formateado con sangría de 2 espacios
  • Validación con mensajes de error claros
  • Minificación para transferencias más rápidas
  • Resaltado de sintaxis XML
  • 100% procesamiento en el navegador

Preguntas Frecuentes

¿Se envían mis datos a un servidor? +
No. El formateador XML usa el DOMParser del navegador. Todo el procesamiento es local y tus datos nunca salen de tu dispositivo.
¿Qué tipos de archivos XML acepta? +
Acepta cualquier archivo XML válido, incluyendo XHTML, SVG, RSS, Atom, SOAP y otros formatos basados en XML.
