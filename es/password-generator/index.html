Generador de Contraseñas

Crea contraseñas aleatorias y seguras usando mayúsculas, minúsculas, números y símbolos. Elige la longitud (6–64 caracteres) y genera varias contraseñas a la vez. Todo sucede en tu navegador con la API Web Crypto — nada se envía a un servidor.

Cómo Usar el Generador de Contraseñas

  1. Paso 1 — Arrastra el control deslizante de longitud para establecer el número de caracteres entre 6 y 64.
  2. Paso 2 — Marca o desmarca los tipos de caracteres: mayúsculas, minúsculas, números y/o símbolos.
  3. Paso 3 — Establece la cantidad, haz clic en "Generar Contraseña" y luego copia individualmente o usa "Copiar todo".

Características

  • Aleatoriedad criptográficamente segura usando la API Web Crypto
  • Longitud personalizable de 6 a 64 caracteres
  • Activa o desactiva mayúsculas, minúsculas, números y símbolos
  • Genera hasta 20 contraseñas a la vez
  • Indicador de fortaleza de contraseña (Débil / Media / Fuerte)
  • Copia con un clic para cada contraseña o todas a la vez
  • 100% en el cliente — las contraseñas nunca salen de tu navegador

Preguntas Frecuentes

¿Qué tan seguras son las contraseñas generadas? +
Las contraseñas se generan usando la API Web Crypto del navegador (crypto.getRandomValues), que produce números aleatorios criptográficamente seguros, el mismo estándar utilizado por software de seguridad y gestores de contraseñas.
¿Se almacenan o transmiten mis contraseñas? +
No. Toda la generación de contraseñas ocurre localmente en tu navegador. Las contraseñas nunca se almacenan, registran ni transmiten a ningún servidor.
¿Qué longitud de contraseña debo usar? +
Para la mayoría de las cuentas en línea, se recomienda un mínimo de 16 caracteres con una mezcla de todos los tipos de caracteres. Para cuentas muy sensibles, usa 20 o más caracteres.
¿Qué mide el indicador de fortaleza? +
El medidor de fortaleza evalúa la longitud y la variedad de caracteres. Una contraseña "Fuerte" tiene al menos 12 caracteres y usa mayúsculas, minúsculas, números y símbolos.
¿Puedo generar varias contraseñas a la vez? +
Sí. Establece el campo Cantidad en cualquier número entre 1 y 20, luego haz clic en Generar. Usa "Copiar todo" para copiar todas las contraseñas generadas al portapapeles de una vez.
