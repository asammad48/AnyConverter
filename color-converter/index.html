Frequently Asked Questions

What is the difference between HEX and RGB? +
HEX and RGB represent the same colors but in different notations. HEX uses a 6-digit hexadecimal code (e.g. #FF5733), while RGB uses decimal values for Red, Green and Blue channels (e.g. rgb(255, 87, 51)).
What is HSL? +
HSL stands for Hue, Saturation, Lightness. It's a more intuitive model for humans — you think in terms of color angle (0–360°), how vivid it is (saturation), and how light or dark it is (lightness).
What is CMYK used for? +
CMYK (Cyan, Magenta, Yellow, Key/Black) is used in print design. While screens use RGB (additive color), printers use CMYK (subtractive color mixing).
What is HSB (also called HSV)? +
HSB (Hue, Saturation, Brightness) — also called HSV (Value) — is similar to HSL but uses Brightness/Value instead of Lightness. It's used in many design applications like Photoshop.
How do I find the HEX code for a color on screen? +
Use your operating system's built-in color picker, browser devtools (inspect element → color swatch), or screenshot and paste tools. Then paste the HEX value here to convert to other formats.
