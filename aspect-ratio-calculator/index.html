How to Use the Aspect Ratio Calculator

  1. Find a ratio: Enter width and height (e.g. 1920 × 1080) and click "Calculate" to get the simplified ratio and decimal.
  2. Scale dimensions: Enter a ratio (e.g. 16:9) and a known width or height. Click "→ Height" or "→ Width" to find the missing dimension.
  3. Presets: Click a preset button to auto-fill common ratios like 16:9 or 4:3.

Frequently Asked Questions

How is the aspect ratio simplified? +
The calculator uses the Greatest Common Divisor (GCD) algorithm to reduce the width and height to their smallest whole-number ratio. For example, 1920:1080 simplifies to 16:9 because both numbers are divisible by 120.
What is the decimal ratio used for? +
The decimal ratio (width ÷ height) is useful for CSS aspect-ratio property and some design tools that accept a single decimal value instead of two numbers. For 16:9 the decimal is 1.7778.
Can I use this for video editing? +
Yes. This tool is commonly used to find the correct height for a thumbnail given a width (or vice versa), to verify that a crop maintains a specific aspect ratio, or to convert between standard video dimensions like 4K (3840×2160) and 1080p (1920×1080).
