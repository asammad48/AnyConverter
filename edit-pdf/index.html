Edit PDF — Add Text & Shapes

Overlay text, border rectangles or yellow highlights onto any PDF page. Specify coordinates in PDF points, build a list of annotations, then download the edited file.

Drop your PDF here or click to browse

Coordinates are in PDF points from the bottom-left corner. A4 = 595×842 pt.

How to Edit a PDF

  1. Step 1 — Upload your PDF (first page dimensions shown).
  2. Step 2 — Choose the annotation type, page, X/Y position and properties.
  3. Step 3 — Click "+ Add Annotation" to queue it. Repeat as needed.
  4. Step 4 — Click "Apply & Download".

Features

  • Three annotation types: text overlay, border rectangle, yellow highlight
  • Custom font size and colour for text
  • Custom width and height for rectangles and highlights
  • Queue multiple annotations before applying
  • 100% browser-based — no server upload

Frequently Asked Questions

How do I find the right coordinates? +
Open the PDF in Adobe Acrobat Reader, enable the coordinate display (View → Show/Hide → Cursor Coordinates), then hover over the target location. Remember PDF coordinates start from the bottom-left — y=0 is the bottom of the page.
Can I edit existing text in the PDF? +
No. This tool overlays new content on top of existing pages — it does not modify or replace existing text. True text editing in PDFs requires dedicated editing software.
