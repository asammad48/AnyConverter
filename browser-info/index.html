Frequently Asked Questions

Is this information private? +
All information is read directly from your browser's JavaScript APIs and displayed locally. Nothing is sent to any server. However, this is the same data that any website you visit can access.
Why does it show the wrong browser? +
Some browsers modify their User Agent string to match another browser for compatibility. Chromium-based browsers like Brave may appear as Chrome. The detection is based on User Agent parsing which can be imprecise.
