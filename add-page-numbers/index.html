Add Page Numbers to PDF

Stamp page numbers onto every page of your PDF. Choose the position (header or footer), font size, starting number, and optional prefix like "Page " or "- ".

How to Add Page Numbers to a PDF

  1. Step 1 — Upload your PDF file.
  2. Step 2 — Choose a position (e.g. Bottom Centre), font size, starting number, and optional prefix.
  3. Step 3 — Click "Add Page Numbers & Download".

Features

  • 6 position options: header and footer, left/centre/right
  • Custom starting number (e.g. start from 5 to skip a cover page)
  • Optional text prefix (e.g. "Page ", "- ", "Pg.")
  • Adjustable font size from 6pt to 30pt
  • 100% browser-based processing

Frequently Asked Questions

Can I start page numbers from 0 or skip the first page? +
Yes. Set the starting number to 0 to begin from zero, or remove page 1 from the count by merging with the Organize PDF tool first and adjusting the starting number accordingly.
What font is used? +
Page numbers are rendered in Helvetica, a clean sans-serif font that is embedded in all standard PDF viewers. No external fonts are required.
