How to Combine Scans into a PDF

  1. Step 1 — Click "Add Images" to select your scanned photos (you can add more in batches).
  2. Step 2 — Remove any unwanted images using the ✕ button.
  3. Step 3 — Choose the output page size.
  4. Step 4 — Click "Create PDF & Download".

Features

  • Add images incrementally — no need to select all at once
  • Thumbnail preview with remove button for each image
  • Three page sizes: original, A4, Letter
  • Images centred and scaled to fit the page
  • 100% browser-based — no server upload

Frequently Asked Questions

What is the difference between Scan to PDF and JPG to PDF? +
Scan to PDF lets you incrementally add images one batch at a time and remove individual images before converting — useful when dealing with scanned documents. JPG to PDF is a simpler single-step conversion.
Can I make the text in scanned images searchable? +
Not directly with this tool. Use the OCR PDF tool after converting to add a searchable text layer to the resulting PDF.
