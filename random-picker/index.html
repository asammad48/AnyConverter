Random Picker Online

Paste a list of items and randomly select any number of them. Items can be separated by newlines or commas. Selected items are highlighted in the pool.

How to Pick Random Items

  1. Step 1 — Paste your list of items into the text area (one per line, or comma-separated).
  2. Step 2 — Select the separator that matches your list format.
  3. Step 3 — Set how many items to pick.
  4. Step 4 — Click Pick! to see the selected items highlighted in the pool.

Features

  • Newline or comma separator
  • Pick any number of items (up to the list size)
  • Selected items displayed as badges
  • Full pool shown with selected items highlighted
  • Uses Fisher-Yates shuffle for uniform randomness

Frequently Asked Questions

Are duplicates possible? +
No. The picker uses a shuffle algorithm that ensures each item can only be selected once per pick. If your list contains duplicate entries they may each be selected independently.
How large can my list be? +
There is no hard limit. Lists with hundreds of items work fine since all processing is done in the browser with JavaScript.
