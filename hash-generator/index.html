Frequently Asked Questions

What is a hash function? +
A hash function takes input data and produces a fixed-size output (the hash). The same input always produces the same hash, but you cannot reverse a hash to get the original input.
What is SHA-256 used for? +
SHA-256 is widely used for data integrity verification, digital signatures, and in blockchain technology. It produces a 256-bit (64 hex character) output.
Is MD5 secure? +
MD5 is considered cryptographically broken and should not be used for security purposes. It is still useful for non-cryptographic uses like checksums and data integrity checks.
Can I verify a file download with a hash? +
Yes. Use File Input mode, drop your downloaded file, and compare the SHA-256 or SHA-512 hash with the checksum provided by the download source. If they match, the file is intact.
Why do hashes look like random letters and numbers? +
Hash outputs are displayed in hexadecimal (base-16) format, using digits 0–9 and letters a–f. Each pair of hex characters represents one byte of the hash output.
