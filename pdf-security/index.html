How to Set PDF Permissions

  1. Step 1 — Upload your PDF.
  2. Step 2 — Set an owner password (required) and optional user password.
  3. Step 3 — Tick or untick the permissions you want to allow.
  4. Step 4 — Click "Apply Security & Download".

Features

  • Owner password required to change permissions
  • Optional user password to restrict opening
  • 5 permission flags: print, copy, modify, annotate, fill forms
  • Compatible with all standard PDF viewers
  • 100% browser-based — no server upload

Frequently Asked Questions

What is the difference between an owner and a user password? +
The user password is required to open the PDF. The owner password is required to change permissions or remove restrictions. Both are set here; only the owner password is mandatory.
Can I remove these restrictions later? +
Yes. Use the Unlock PDF tool and enter the owner password to remove all restrictions and produce an unrestricted copy.
