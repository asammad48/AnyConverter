Features

  • Works with scanned PDFs and image files (JPG, PNG)
  • 10 language options including English, Spanish, French, German, Chinese
  • Multi-page PDF support — extracts text from all pages
  • Powered by Tesseract.js — runs entirely in the browser
  • Copy text button for quick clipboard access

Frequently Asked Questions

Why is the OCR slow? +
Tesseract.js runs entirely in your browser using WebAssembly. The first run downloads the language data (~8 MB for English) and processes each page individually. Multi-page PDFs take longer. For faster results, extract the pages you need first using the Extract Pages tool.
My text is already selectable in the PDF — do I need OCR? +
No. OCR is only needed for scanned PDFs where the content is essentially an image. If you can select text in your PDF viewer, the text layer already exists and you can copy it directly.
