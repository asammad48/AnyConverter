Security at AnyConverter

Last updated: May 26, 2026

Security and privacy are core to how AnyConverter is built — not bolted on as an afterthought. Here's how we protect your data and our platform.

🔒

No Server Uploads

All tool processing happens in your browser. Files never reach our servers.

🔐

HTTPS Everywhere

Every page is served over TLS. All connections are encrypted in transit.

🚫

No Accounts

No usernames, passwords, or session tokens. Nothing to steal.

🛡️

CDN Protection

Served via Cloudflare with DDoS protection and edge caching.

Client-Side Architecture

The single most important security property of AnyConverter is that all tool processing happens inside your browser. When you use our PDF merger, JSON formatter, image converter, or any other tool, your data is processed by JavaScript running on your own device.

This means:

  • We cannot see your files, even if we wanted to
  • There is no server-side database storing your content
  • Closing the browser tab immediately removes all traces of your data
  • Most tools continue working offline after the page loads

HTTPS & Transport Security

All traffic to and from AnyConverter is encrypted using TLS 1.2 or higher. We use HTTP Strict Transport Security (HSTS) to prevent downgrade attacks. Our SSL certificates are managed automatically through Cloudflare.

Content Security Policy

AnyConverter implements a strict Content Security Policy (CSP) to prevent cross-site scripting (XSS) attacks. We only load scripts and resources from trusted sources (our own domain, Google Fonts, and explicitly permitted CDN libraries).

Third-Party Libraries

Where we use third-party JavaScript libraries (pdf-lib, PapaParse, highlight.js), we load them from trusted CDNs with Subresource Integrity (SRI) hashes where possible, ensuring the code hasn't been tampered with.

No Account Data to Breach

Because AnyConverter has no user accounts, there is no database of usernames, emails, or passwords that could be breached. We cannot lose data we never collected.

Infrastructure

AnyConverter is a static website deployed on Cloudflare Pages, one of the most secure and performant static hosting platforms available. Cloudflare provides automatic DDoS mitigation, WAF rules, and edge caching at 200+ global locations.

🔍 Responsible Disclosure

If you discover a security vulnerability in AnyConverter, we encourage you to report it responsibly. We take all security reports seriously and will respond promptly.

Please report security issues to: security@anyconverter.io

Please include a clear description of the vulnerability, steps to reproduce it, and potential impact. We kindly ask that you do not publicly disclose the issue until we have had a reasonable time to investigate and respond (typically within 72 hours).

We do not currently offer a bug bounty programme, but we will publicly acknowledge your contribution if you wish.

Questions
