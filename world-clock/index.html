World Clock

See the live current time in major cities across every time zone. Add or remove cities, and check if it's day or night at a glance. Updates every second.

How to Use the World Clock

  1. Step 1 — The clock starts with 10 default cities, all ticking live.
  2. Step 2 — Select a city from the dropdown and click "+ Add City" to add it.
  3. Step 3 — Click the ✕ on any card to remove a city.
  4. Night mode: Cards with a dark background indicate it is currently night in that city (between 20:00 and 06:00).

Features

  • Live time updates every second
  • 40+ cities and time zones available
  • Day/night indicator on each clock card
  • UTC offset displayed for each city
  • Add and remove cities on the fly
  • Handles DST automatically via browser timezone API

Frequently Asked Questions

Does the clock update automatically? +
Yes. The time on each card refreshes every second using JavaScript's setInterval. There is no need to reload the page.
Does it handle Daylight Saving Time? +
Yes. The tool uses the browser's built-in Intl.DateTimeFormat API with IANA timezone identifiers (e.g. America/New_York), which automatically accounts for DST transitions.
Why does a card appear dark? +
Cards with a dark background indicate the local time in that city is between 20:00 (8 PM) and 06:00 (6 AM) — considered night-time hours.
