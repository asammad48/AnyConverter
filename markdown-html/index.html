Frequently Asked Questions

      

        
What is Markdown? +
Markdown is a lightweight markup language that uses plain text formatting syntax. It was designed to be easy to write and read, and converts cleanly to HTML.

        
What is GitHub Flavored Markdown? +
GitHub Flavored Markdown (GFM) is a superset of standard Markdown that adds features like tables, task lists, strikethrough text, and fenced code blocks with syntax highlighting.

        
How do I add a link in Markdown? +
Use [link text](URL) syntax. For example, [AnyConverter](https://anyconverter.io) creates a clickable link.

        
How do I add an image in Markdown? +
Use ![alt text](image-url) — similar to a link but with an exclamation mark at the start. The alt text is shown if the image fails to load.

        
Can I use the HTML output on my website? +
Yes. The generated HTML is clean and ready to paste into any webpage. You may want to add your own CSS styles, as this tool does not include a stylesheet in the downloaded file.
