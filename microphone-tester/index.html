How to Use

  1. Click Start Mic Test — your browser will request microphone permission.
  2. Grant access and the live level bar will start moving as you speak.
  3. The meter turns yellow when loud and red when clipping.
  4. Peak level and device name are shown automatically.

Features

  • Real-time animated audio level bar
  • Approximate dB level display
  • Peak level tracking
  • Shows microphone device name
  • No audio is recorded or uploaded — 100% local

Frequently Asked Questions

Is my audio recorded? +
No. The microphone input is processed entirely in your browser using the Web Audio API. No audio is recorded, stored, or transmitted anywhere.
The bar isn't moving — what should I do? +
Check that your microphone is set as the default input device in your OS settings. Make sure the browser has microphone permission (look for a mic icon in your browser's address bar). Try speaking louder or moving closer to the mic.
