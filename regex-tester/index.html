How to Use the Regex / Log Extractor

  1. Enter a pattern — Type a regex in the pattern bar, or click a Quick Pattern from the sidebar to load it instantly.
  2. Paste log text — Paste your log content into the text area. Matches highlight in real time with color-coded capture groups.
  3. Export results — Click → CSV or → JSON to download all extracted matches. Groups are included as separate columns.

  • Live match highlighting as you type

  • Export all matches to CSV or JSON

Frequently Asked Questions

What does the g flag do? +
The g (global) flag finds all matches in the string. Without it, only the first match is returned.
How do capture groups work? +
Wrap parts of your pattern in parentheses () to create capture groups. Each group is extracted separately and shown in a different color. Named groups use (?<name>...) syntax.
How do I export matches? +
Click → CSV to download a spreadsheet with one row per match and columns for each capture group. Click → JSON to get a JSON array of all matches.
What regex flavor does this use? +
This tester uses JavaScript's built-in RegExp engine (ECMAScript regex), which supports lookaheads, lookbehinds, named groups, Unicode, and all standard quantifiers.
Can I test against large log files? +
Yes. Use the Upload .log button to load a file directly from disk. The tool processes it entirely in your browser — nothing is sent to any server.
