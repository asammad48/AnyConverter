Frequently Asked Questions

What is XML formatting? +
XML formatting adds indentation and line breaks to XML so it is easier to read. Raw or minified XML is often hard to read without proper formatting.
Is my XML data private? +
Yes. All XML processing happens in your browser. Your data is never sent to any server.
What errors does the validator detect? +
The validator uses the browser's built-in DOMParser to detect XML syntax errors including unclosed tags, mismatched elements, and invalid characters.
Can I minify XML? +
Yes. The Minify button collapses your XML to a single line with no whitespace, reducing file size for network transfer.
What is the difference between XML and JSON? +
XML uses tags to describe data structure and supports attributes, namespaces, and comments. JSON is simpler and more compact, commonly used in modern APIs. Both are widely used for data exchange.
