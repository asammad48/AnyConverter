JSON ↔ YAML Konverter

Konverter mellem JSON og YAML øjeblikkeligt. Indsæt din JSON for at få ren YAML-output, eller konverter YAML tilbage til formateret JSON. Validerer syntaks mens du skriver.

Reklame

Sådan bruges JSON til YAML Konverter

  1. Vælg konverteringsretning: JSON→YAML eller YAML→JSON ved hjælp af tilstandsknapperne.
  2. Indsæt din JSON eller YAML i inputområdet (eller klik på "Indlæs Eksempel" for en prøve).
  3. Det konverterede output vises øjeblikkeligt. Klik "Kopiér Output" for at kopiere resultatet.

Funktioner

  • Tovejskonvertering — JSON til YAML og YAML til JSON
  • Realtidskonvertering mens du skriver
  • Syntaksvalidering med klare fejlmeddelelser
  • 2-mellemrums indrykning i YAML-output
  • Indlæs eksempeldata med ét klik
  • 100% klientside — dine data forlader aldrig browseren

Ofte stillede spørgsmål

Hvad er YAML?

YAML (YAML Ain't Markup Language) er et menneskevenligt dataserialiseringsformat. Det bruger indrykning og minimal tegnsætning, hvilket gør det nemmere at læse end JSON. Det bruges bredt til konfigurationsfiler (Docker, Kubernetes, GitHub Actions).

Er al JSON gyldig YAML?

Ja — JSON er teknisk set en delmængde af YAML 1.2. Ethvert gyldigt JSON-dokument er også gyldigt YAML. YAML understøtter dog yderligere funktioner som kommentarer og flerlinjede strenge, som JSON ikke har.

Understøtter YAML kommentarer?

Ja. YAML understøtter kommentarer ved hjælp af #-tegnet. JSON understøtter slet ikke kommentarer. Når YAML konverteres til JSON, kasseres kommentarer.

Hvilken indrykning bruger konverteren?

Konverteren genererer YAML med 2-mellemrums indrykning, som er den mest almindelige konvention for YAML-filer i moderne værktøjer.

Sendes mine data til en server?

Nej. Dette værktøj kører helt i din browser ved hjælp af js-yaml-biblioteket. Dine data uploades aldrig til nogen server, hvilket gør det sikkert til følsomme konfigurationsdata.
