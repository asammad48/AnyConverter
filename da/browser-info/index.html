Ofte Stillede Spørgsmål

Er disse oplysninger private? +
Al information læses direkte fra din browsers JavaScript API'er og vises lokalt. Intet sendes til nogen server. Disse er dog de samme data som ethvert websted du besøger kan tilgå.
Hvorfor viser den den forkerte browser? +
Nogle browsere ændrer deres User Agent-streng for at matche en anden browser af kompatibilitetshensyn. Chromium-baserede browsere som Brave kan fremstå som Chrome. Genkendelsen er baseret på User Agent-parsing, som kan være upræcis.
