Ofte Stillede Spørgsmål

      

        
Hvad er Markdown? +
Markdown er et letvægts markup-sprog der bruger formatering i ren tekst. Det er designet til at være nemt at skrive og læse, og konverteres rent til HTML.

        
Hvad er GitHub Flavored Markdown? +
GitHub Flavored Markdown (GFM) er et supersæt af standard Markdown der tilføjer funktioner som tabeller, opgavelister, overstreget tekst og afgrænsede kodeblokke med syntaksfremhævning.

        
Hvordan tilføjer jeg et link i Markdown? +
Brug [linktekst](URL) syntaks. For eksempel opretter [AnyConverter](https://anyconverter.io) et klikbart link.

        
Kan jeg bruge HTML-outputtet på min hjemmeside? +
Ja. Det genererede HTML er rent og klar til at indsætte på enhver webside. Du vil måske tilføje dine egne CSS-stilarter, da dette værktøj ikke inkluderer et stylesheet i den downloadede fil.
