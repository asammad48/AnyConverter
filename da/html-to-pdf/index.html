HTML til PDF Konverter

Indsæt HTML-kode i editoren, se live forhåndsvisning i browseren, og konverter det til en flersiders A4 PDF. Understøtter inline CSS, tabeller, overskrifter og base64-billeder.

Reklame

Sådan Konverterer Du HTML til PDF

  1. Trin 1 — Indsæt din HTML-kode i editoren (et eksempel er forudindlæst).
  2. Trin 2 — Forhåndsvisning opdateres live mens du skriver.
  3. Trin 3 — Vælg stående eller liggende retning.
  4. Trin 4 — Klik "Konverter til PDF og Download".

Funktioner

  • Live HTML-forhåndsvisning i en iframe
  • Flersiders A4-output — langt indhold deles over sider
  • Stående og liggende retning
  • Understøtter inline CSS, tabeller, billeder (base64) og webskrifttyper
  • Drevet af jsPDF + html2canvas — fuldt browserbaseret

Ofte Stillede Spørgsmål

Kan jeg bruge ekstern CSS eller Google Fonts? +
Eksterne stilark og webskrifttyper fra eksterne domæner er blokeret i iframe-sandkassen. Brug inline <style>-tags til CSS og base64-kodede skrifttyper til brugerdefineret typografi.
Kan jeg konvertere en webside-URL til PDF? +
Ikke direkte. Du kan bruge browserens "Vis sidekilde" til at kopiere HTML'en, indsætte den her og konvertere. Dog vil eksterne ressourcer som CSS og billeder ikke indlæses medmindre de er inline.
