Sådan konverterer du CSV til SQL

  1. Indsæt dit CSV — Indsæt CSV-indholdet eller upload en .csv-fil. Vælg skilletegn efter behov.
  2. Konfigurer kolonnetyper — Gennemgå de auto-detekterede kolonnetyper (INT, VARCHAR, DATE osv.) og juster dem efter behov.
  3. Generer SQL — Indstil tabelnavn, vælg dialekt og klik "Generer SQL". Kopier eller download resultatet.

Funktioner

  • Automatisk detektering af kolonnetyper
  • MySQL, PostgreSQL og SQLite dialekter
  • Genererer CREATE TABLE og INSERT
  • Batch INSERT-understøttelse
  • NULL for tomme celler
  • 100% behandling i browseren

Ofte stillede spørgsmål

Hvilke SQL-dialekter understøttes? +
MySQL (backtick-citering og AUTO_INCREMENT), PostgreSQL (dobbeltcitering af identifikatorer, SERIAL) og SQLite (INTEGER PRIMARY KEY AUTOINCREMENT).
Hvordan håndteres tomme celler? +
Tomme celler i CSV konverteres til NULL i SQL INSERT-sætningerne, som er den standard SQL-repræsentation af en manglende værdi.
Escapes enkeltcitater i data? +
Ja. Enkeltcitater i strengværdier escapes ved at fordoble dem (f.eks. O'Brien bliver O''Brien), hvilket er standard SQL-strengescaping.
