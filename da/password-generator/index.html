Adgangskodegenerator

Opret stærke, tilfældige adgangskoder med store bogstaver, små bogstaver, tal og symboler. Vælg din længde (6–64 tegn) og generer flere adgangskoder på én gang. Al generering sker i din browser ved hjælp af Web Crypto API — intet sendes til en server.

Reklame

Sådan bruges Adgangskodegeneratoren

  1. Trin 1 — Træk længdeskyderne for at angive det ønskede antal tegn mellem 6 og 64.
  2. Trin 2 — Markér eller fjern markeringen af de tegntyper, du ønsker: store bogstaver, små bogstaver, tal og/eller symboler.
  3. Trin 3 — Angiv antallet, klik på "Generer adgangskode", og kopiér individuelle adgangskoder eller brug "Kopiér alle".

Funktioner

  • Kryptografisk sikker tilfældighed ved hjælp af Web Crypto API
  • Tilpasselig længde fra 6 til 64 tegn
  • Slå store bogstaver, små bogstaver, tal og symboler til/fra uafhængigt
  • Generer op til 20 adgangskoder på én gang
  • Adgangskodestyrkemåler (Svag / Middel / Stærk)
  • Kopiér med ét klik for hver adgangskode eller alle på én gang
  • 100% klientbaseret — adgangskoder forlader aldrig din browser

Ofte stillede spørgsmål

Hvor sikre er de genererede adgangskoder? +
Adgangskoder genereres ved hjælp af browserens indbyggede Web Crypto API (crypto.getRandomValues), som producerer kryptografisk sikre tilfældige tal — den samme standard, der bruges af sikkerhedssoftware og adgangskodeadministratorer.
Gemmes eller overføres mine adgangskoder? +
Nej. Al adgangskodegenerering sker lokalt i din browser. Adgangskoderne gemmes, logges eller overføres aldrig til nogen server. Når du lukker eller genindlæser siden, er de væk.
Hvilken adgangskodelængde bør jeg bruge? +
For de fleste onlinekonti anbefales mindst 16 tegn med en blanding af alle tegntyper. For meget følsomme konti som e-mail eller bank bør du bruge 20 eller flere tegn.
Hvad måler styrkeindikatoren? +
Styrkemåleren evaluerer længde og tegnvariation. En "Stærk" adgangskode er mindst 12 tegn lang og bruger en blanding af store bogstaver, små bogstaver, tal og symboler.
Kan jeg generere flere adgangskoder på én gang? +
Ja. Indstil feltet Antal til et tal mellem 1 og 20, og klik derefter på Generer. Brug "Kopiér alle" for at kopiere alle genererede adgangskoder til udklipsholderen på én gang.
