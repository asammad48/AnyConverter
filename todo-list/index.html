    How to Use the To-do List

    1. Step 1 — Type a task in the input box and click Add or press Enter.
    2. Step 2 — Check the checkbox next to a task to mark it complete.
    3. Step 3 — Use the filter buttons to view All, Active, or Completed tasks.
    4. Step 4 — Click Clear completed to remove all done tasks at once.

    Features

    • Saves to localStorage — tasks persist between sessions
    • Filter by All, Active, or Completed
    • Clear completed tasks with one click
    • Item counter shows remaining active tasks
    • Works offline — no internet connection needed after page load

    Frequently Asked Questions

    Will my tasks be saved if I close the browser? +
    Yes. Tasks are stored in your browser's localStorage and persist between sessions on the same device and browser. Clearing browser data or using private/incognito mode will erase them.
    Can I sync tasks across devices? +
    Not currently — localStorage is device and browser-specific. For cross-device sync you would need an account-based service.
