How to Use

  1. Click anywhere on the page to make sure it has focus.
  2. Press any key — it will light up purple on the keyboard diagram.
  3. The key log shows each keydown and keyup event with the key code.
  4. Click Reset All to clear all highlighted keys.

Features

  • Visual QWERTY keyboard layout with all standard keys
  • Keys light up on press and release
  • Key event log with keydown / keyup indicators
  • Detects modifier keys: Shift, Ctrl, Alt, Win/Cmd
  • Works with any physical keyboard layout

Frequently Asked Questions

Why doesn't a key show up when I press it? +
The key may be physically damaged or the connection is broken. Some special keys (like Fn on laptops) are handled by hardware and never send a keycode to the browser.
How do I test for ghost keys? +
Hold down multiple keys simultaneously. If a key registers without you pressing it, that is a ghost key — a limitation of the keyboard's hardware matrix. Higher quality keyboards (with N-Key Rollover) prevent this.
