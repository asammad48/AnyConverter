Frequently Asked Questions

What scale should I use for print-quality images? +
Use 3× for print-quality output (~216 DPI). For web use 2× is sufficient. 1× gives screen-resolution images useful for previews.
Can I convert just one page? +
The tool converts all pages, but you can choose to save only the specific page images you want. Alternatively, use the Extract Pages tool first to create a single-page PDF, then convert it here.
