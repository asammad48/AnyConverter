Frequently Asked Questions

What makes a password strong? +
A strong password has at least 12 characters, includes uppercase and lowercase letters, numbers, and special characters, and is not a commonly used word or phrase. This tool checks all these criteria simultaneously.
Is my password stored anywhere? +
No. This tool runs entirely in your browser using JavaScript. Your password is never sent to any server and never stored anywhere. You can even disconnect from the internet before typing your password.
What counts as a special character? +
Any character that is not a letter (a-z, A-Z) or digit (0-9) is considered special. Examples include: ! @ # $ % ^ & * ( ) _ - + = [ ] { } | \ : ; " ' < > , . ? /
What does "common password" mean? +
The tool checks your password against a list of the most frequently used passwords (like "password", "123456", "qwerty"). Using a common password is extremely risky because attackers always try these first.
