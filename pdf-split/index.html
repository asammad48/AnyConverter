How to Split a PDF

  1. Upload your PDF — Drop the PDF file or click to select it. The tool shows the total page count.
  2. Choose split mode — Extract specific pages by range, split every N pages, or extract each page individually.
  3. Download — Click Split PDF and download each resulting file.

Frequently Asked Questions

How do I specify page ranges? +
Use commas to separate individual pages and dashes for ranges. Example: 1,3,5-8,12 extracts pages 1, 3, 5, 6, 7, 8, and 12.
Can I extract just one page from a PDF? +
Yes. Use Extract specific pages mode and enter just the page number you want (e.g. "5" to get only page 5).
Are my PDFs uploaded to a server? +
No. All splitting happens locally in your browser using the pdf-lib JavaScript library. Your files never leave your device.
What happens to the page quality when splitting? +
pdf-lib copies pages without re-processing them, so quality is perfectly preserved. Text, images and graphics remain identical to the original.
Can I split a large PDF? +
Yes. There is no hard size limit. Very large PDFs may take a moment to process in the browser, but all splitting is done client-side without any upload.
