Optimize & Compress PDF

Shrink your PDF by stripping unreferenced objects and optionally wiping metadata. The tool shows you the before and after file sizes so you can see the saving.

How to Optimize a PDF

  1. Step 1 — Upload your PDF.
  2. Step 2 — Optionally tick "Strip metadata" for additional savings.
  3. Step 3 — Click "Optimize & Download" to get the smaller PDF.

Features

  • Removes unreferenced/orphaned PDF objects
  • Uses object streams for tighter packing
  • Optional metadata stripping (title, author, keywords, producer)
  • Shows before & after file size comparison
  • 100% browser-based — file never leaves your device

Frequently Asked Questions

How much will this reduce my PDF? +
It depends on the PDF. Files generated by incremental editing (many saves) benefit most. PDFs already well-optimised may see little or no reduction — the tool will tell you.
Does this reduce image quality? +
No. This tool restructures the PDF without re-encoding images. Image quality is completely preserved. For image downscaling you'd need a server-side tool.
