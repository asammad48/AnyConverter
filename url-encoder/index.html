How to Encode or Decode a URL

  1. Choose mode — Select Encode Component for query parameters, Encode Full URL to preserve URL structure, or Decode to reverse encoding.
  2. Enter your text or URL — Paste the string you want to encode or decode into the input area.
  3. Click Convert — Copy the output or use Swap to move the output back to input for chaining.

Features

  • Encode URL components (encodeURIComponent)
  • Encode full URLs (preserves ://&=?#)
  • Decode percent-encoded URLs
  • Swap input and output
  • Live conversion on input
  • 100% browser-based, no server

Frequently Asked Questions

What is URL encoding? +
URL encoding converts special characters in a URL to their percent-encoded equivalents. For example, a space becomes %20 and & becomes %26. This ensures URLs are transmitted correctly.
When should I use Encode Component vs Encode Full URL? +
Use Encode Component for individual query parameter values — it encodes everything including / and ?. Use Encode Full URL when you want to keep the URL structure intact but encode unsafe characters.
What characters are encoded? +
Encode Component encodes all characters except letters, digits, and -_.~. Encode Full URL also preserves :/?#[]@!$&'()*+,;= which are valid URL characters.
What is %20 vs +? +
%20 is the standard percent-encoding for a space. The + sign also represents a space but only in query strings (form data). This tool uses %20 for standard URL encoding.
Is URL decoding safe? +
Yes. URL decoding is safe and reversible. All processing happens in your browser and no data is sent to any server.
